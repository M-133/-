**التفاصيل:**
- قم بإنشاء متغير لتخزين الرقم.
- استخدم دالة `cin` لأخذ الرقم من المستخدم.

- بعد انتهاء الحلقة، قم بمقارنة مجموع المقسومات الصحيحة مع الرقم الأصلي.
- إذا كانت النتيجة متساوية، فإن الرقم مثالي. وإلا فهو غير مثالي.
- عرض النتيجة باستخدام دالة `cout`.

**مثال:**
إذا أدخل المستخدم الرقم 6، فإن البرنامج سيقوم بتحديد أن الرقم مثالي لأن مجموع مقسوماته الصحيحة (1 + 2 + 3) يساوي الرقم نفسه.
